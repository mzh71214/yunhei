1.导入数据库文件 install.sql导入到数据库
2.修改config.php里的数据库信息
3.默认后台地址 /admin/ 可以直接改文件名修改后台地址
4.后台用户名和密码统一为 admin
5.登录后请修改密码！
6.如需更换首页背景，覆盖文件即可。背景图片位置 /images/bj.png

7.调用api接口【 http://你的域名/query.php?qq=】

8. results.php 这个文件也是查询QQ用的跟上面那个一样用 ?qq=，不过用不上，API建议用 7 的方法，这个文件删不删都无所谓

这个源码是网上开源的，不保证绝对安全，有能力的自己优化。

环境建议：Nginx1.16   PHP7.2(只推荐这个，因为我用其他版本发现有报错)   MySQL5.6以上
